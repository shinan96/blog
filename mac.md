- Command（或 cmd）：⌘
- Shift ：⇧
- Option（或 Alt）：⌥
- Control（或 ctrl）：⌃



#### 快捷键

| 快捷键               | 功能             |
| -------------------- | ---------------- |
| ctrl + ⌘ + F         | 全屏             |
| ⌘ + N （桌面空白处） | 打开Finder       |
| ⌘+option+ space      |                  |
| ⌘+shift+g            | 当前文件夹中搜索 |
| ⌘ + ~                | 多窗口切换       |
| ⌘+⌥+C                | 拷贝文件夹路径   |







#### 谷歌浏览器快捷键

| 快捷键        | 功能                     |
| ------------- | ------------------------ |
| ⌘+N           | 打开新窗口               |
| ⌘+shift+N     | 新建无痕窗口             |
| ⌘+T           | 打开新的标签页           |
| ⌘+W           | 关闭当前页               |
| ⌘+shift+W     | 关闭当前窗口。           |
| ⌘+shift+T     | 重新打开上次关闭的标签页 |
| ⌘+Option+方向 | 切换标签页               |
