#### 快捷键

| 快捷键        | 功能                     |
| ------------- | ------------------------ |
| ⌘+N           | 打开新窗口               |
| ⌘+shift+N     | 新建无痕窗口             |
| ⌘+T           | 打开新的标签页           |
| ⌘+W           | 关闭当前页               |
| ⌘+shift+W     | 关闭当前窗口。           |
| ⌘+shift+T     | 重新打开上次关闭的标签页 |
| ⌘+Option+方向 | 切换标签页               |
