## 组件传值方法


### prop/$emit
### .sync
```
父组件通过props传值给子组件，子组件$emit,update修改父组件的值
```

``` javascript
父组件
<child :msg='value' ></child>

子组件
this.$emit('update:msg', '子组件的值')
```
这种方式违背了vue数据流单向流动，子组件直接修改了父组件的数据
