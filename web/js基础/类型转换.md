##  类型转换

### 隐私转换
* 转Boolean
* 转数字
* 转字符串

#### 转Boolean
```
在条件判断中，除了undefined, null, 0, -0, '', false, NaN， 其他的都转成true。
```

### 四则运算符
* 运算中其中一方为字符串，那么就会把另一方也转换为字符串
* 如果一方不是字符串或者数字，那么会将它转换为数字或者字符串
```js
1 + '1'     // '11'
true + true // 2
4 + [1,2,3] // "41,2,3"
```
### 字符串转数组
```
let str = '123abc123asffa'

1. str.splite('')
2. [...str]
```
