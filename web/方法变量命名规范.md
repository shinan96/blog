### method方法命名

1. 请求类：getName， postName；

2. 页面跳转：jumpRouter;

3. 常用 

   add/delete，添加/删除
   insert/delete，插入/删除
   start/stop，开始/停止
   begin/end，开始/结束
   send/receive，发送/接收
   get/set，取出/设置

   up/down，向上/向下
   show/hide，显示/隐藏
   open/close，打开/关闭
   increment/decrement，增加/减少
   lock/unlock，锁/解锁

   表单提交：submit、send
   表单增删改查：add、delete、update、search、reset
   上传附件：upload
   关闭打开弹窗：open/close
   检查：check
   
4. Api

   动词+name + Api
